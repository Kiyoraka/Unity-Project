# Simple 2D Main Menu Script for Unity

## Overview
This repository hosts a script specifically designed for creating a Simple 2D Main Menu in Unity-based games. It's tailored to be user-friendly and easily customizable to fit the unique needs of your game.

## Installation
Adding the Script to Your Project:

1. Drag the Simple2DMainMenu.cs script into an Empty Game Object within your Unity project's hierarchy. This step sets up the foundational structure for your main menu.
Configuring Buttons:

2. In your Unity project, select the buttons intended for your main menu.
Drag these buttons into the corresponding fields in the script inspector. This links your buttons to the menu, enabling navigation functionality.

## Customization

1. After integrating the buttons, you can customize the menu's appearance and behavior to align with your game's design.

2. Experiment with different settings and styles to create a unique user experience.

## Testing

1. Test the main menu within the Unity editor to ensure all interactions and navigations are functioning as expected.
